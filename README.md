# Tugas-Besar-IF4802-Kelompok_06
Program Multi-Linked List (1-N) relasi Paper &amp; Keyword secara modular. Paper.h dan Keyword.h mendefinisikan ADT, sedangkan file .cpp menangani logika operasi (Create, Insert, Show, Dll). Entry point di main.cpp memisahkan akses antarmuka: main_admin untuk manajemen data (CRUD) dan main_user untuk fitur studi kasus pencarian data.
